AutoScraper Documentation

Advanced Google scraper and business contact finder

Overview

AutoScraper is a powerful tool designed to automate the process of searching Google for businesses, extracting contact information, and sending personalized emails. It features advanced techniques to bypass Google's security measures, making it an effective solution for lead generation and outreach campaigns.

The tool is built with a focus on flexibility, performance, and ease of use, offering both a command-line interface and a web interface for different user preferences.

Key Features

Google Search Scraping

Automatically search Google for businesses based on keywords and extract URLs from search results.

Implements advanced techniques to bypass Google's security measures and avoid being blocked.

Contact Information Extraction

Scrape websites to find email addresses, phone numbers, and other contact information.

Intelligent parsing to identify the most relevant contact details for businesses.

Email Automation

Send personalized emails to extracted contacts using customizable templates.

Track email sending progress and manage outreach campaigns efficiently.

Web Interface

User-friendly web interface for configuring and monitoring scraping jobs.

View and download results in various formats (CSV, JSON).

Command-Line Interface

Powerful CLI for automation and integration with other tools and scripts.

Rich output with progress indicators and detailed logging.

Docker Support

Easy deployment with Docker and docker-compose.

Consistent environment across different platforms.

Architecture

AutoScraper follows a modular architecture with clear separation of concerns. The main components include:

  • GoogleScraper: Core class for searching Google and extracting URLs
  • EmailSender: Handles email composition and sending
  • Web Interface: FastAPI-based web application for user interaction
  • CLI: Command-line interface for scripting and automation

Installation

Using Python

# Clone the repository
git clone https://github.com/yourusername/autoscraper.git
cd autoscraper

# Run the installation script
chmod +x install.sh
./install.sh

Using Docker

# Clone the repository
git clone https://github.com/yourusername/autoscraper.git
cd autoscraper

# Build and run with Docker Compose
docker-compose up -d

Usage

Web Interface

The web interface provides an intuitive way to use AutoScraper. After installation: 

# Start the web server
python -m autoscraper.app

# Or use the convenience script
python run.py web

Then open your browser and navigate to http://localhost:8000.

Command-Line Interface

The CLI offers powerful options for automation and scripting: 

# Basic search
autoscraper search "dentists amsterdam" --pages 3

# Search and send emails
autoscraper search "lawyers new york" --pages 5 --send-emails --email-template introduction_email

# List available email templates
autoscraper templates

API Reference

Web API Endpoints

Endpoint Method Description
/scrape POST Start a new scraping job
/results GET View scraping results
/download/{filename} GET Download results as CSV
/send-email POST Send an email to a specific contact
/email-templates GET List available email templates

Python API

You can also use AutoScraper as a Python library in your own projects: 

import asyncio
from autoscraper import GoogleScraper, EmailSender

async def main():
    # Initialize the scraper
    scraper = GoogleScraper()
    
    # Search for businesses
    await scraper.search_google("dentists amsterdam", num_pages=3)
    
    # Process the URLs to extract contact information
    await scraper.process_urls()
    
    # Save the results
    scraper.save_results("dentists_amsterdam")
    
    # Send emails (optional)
    scraper.send_emails_to_companies("introduction_email")

# Run the async function
asyncio.run(main())

Email Templates

AutoScraper comes with several pre-defined email templates that you can customize:

  • introduction_email.txt: A general introduction to your services
  • partnership_email.txt: Proposal for potential business partnerships
  • service_offer_email.txt: Special offer for your services
  • event_invitation_email.txt: Invitation to an event or webinar

Templates use placeholders like {company_name}, {sender_name}, etc., which are automatically replaced with actual values when sending emails.

Troubleshooting

Important: Google may block automated scraping attempts. AutoScraper implements various techniques to avoid detection, but there's no guarantee these will always work as Google regularly updates its security measures.

Common Issues

  • No results found: Google might be blocking your requests. Try increasing the delay between requests or using a proxy.
  • Email sending fails: Check your SMTP settings and ensure you're using an app password if using Gmail.
  • Rate limiting: If you're getting 429 errors, you're being rate limited. Increase the delay between requests.
Tip: Check the debug directory for HTML responses from Google to diagnose scraping issues.
