# Images Directory

This directory contains visual assets for the AutoScraper documentation.

## Logo

To add a proper logo, create an image file named `logo.png` and place it in this directory.
The recommended size is 500x500 pixels with a transparent background.

## Screenshots

Project screenshots can be added to this directory with descriptive names, for example:
- `web_interface.png` - Screenshot of the web interface
- `cli_example.png` - Example of using the command-line interface
- `results_page.png` - Screenshot of the results page

These images can then be referenced in the HTML documentation.
